## A Comparative Study of MLP, Random Forest, and Naive Bayes  

## Project Overview
This project was developed as a final assignment for the Machine Learning course.  
The objective of this study is to perform a comparative analysis of supervised machine learning algorithms to classify cardiac catheterization (Cath) outcomes based on clinical features.

The project implements multiple classification models and evaluates their performance to determine the most effective algorithm for predicting heart disease outcomes.

---

## Problem Statement
Cardiac catheterization (Cath) is a medical procedure used to diagnose heart disease.  
This project aims to build a predictive model that classifies Cath outcomes using patient clinical data.

---

## Dataset Features

### Input Features
- Age  
- Sex  
- BMI (Body Mass Index)  
- HTN (Hypertension)  
- DM (Diabetes Mellitus)  

### Target Variable
- Cath  

---

## Machine Learning Models
The following supervised learning models were implemented and compared:

- Multi-Layer Perceptron (MLP)  
- Tuned MLP (Hyperparameter Optimization)  
- Random Forest  
- Naive Bayes  

---

## Model Performance Comparison

| Model           | Accuracy (%) |
|----------------|--------------|
| MLP (Tuned)    | 80.33       |
